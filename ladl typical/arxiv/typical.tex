\section{The interpretation}

One object $\mathcal{T}$ of terms.  Generating morphisms
\begin{itemize}
  \item $\mathsf{S} : \mathsf{I} \rightarrow \mathcal{T}$
  \item $\mathsf{K} : \mathsf{I} \rightarrow \mathcal{T}$
  \item $\mathsf{()} : \mathcal{T} \otimes \mathcal{T} \rightarrow \mathcal{T}$ // application
  \item $\delta : \mathcal{T} \rightarrow \mathsf{I}$ // delete a subterm
  \item $\Delta : \mathcal{T} \rightarrow \mathcal{T} \otimes \mathcal{T}$ // duplicate a subterm
  \item $\mathsf{R} : \mathcal{T} \rightarrow \mathcal{T}$ // Reduction context
\end{itemize}
Generating rewrites
\begin{itemize}
  \item $\forall\; x,y,z\; \mathsf{R}((x y) z) \rightarrow \mathsf{R}(\mathsf{R}(x y) z)$
  \item $\forall\; x,y\; \mathsf{R}(\mathsf{R}(K x) y) \rightarrow x \otimes \delta(y)$
  \item $\forall\; x,y,z\; \mathsf{R}(\mathsf{R}(\mathsf{R}(S x) y) z) \rightarrow \mathsf{R}((-_{1} -_{2}) (-_{3} -_{4})) \circ (T \otimes swap \otimes T) \circ (T \otimes T \otimes \Delta) (x, y, z)$
    i.e. $\mathsf{R}((x z) (y z))$
\end{itemize}

We can take the 2-category to be Poset as before.

This is a completely untyped combinator calculus.  

\subsection{Semantics}

To get types, we
can consider sets of terms that satisfy a proposition:

\begin{itemize}
  \item $\meaningof{\top} = \mathcal{T}$
  \item $\meaningof{\bot} = \empty$
  \item $\meaningof{S} = \{ S \}$
  \item $\meaningof{K} = \{ K \}$
  \item $\meaningof{ A 1 B } = \{ t \in \mathcal{T} | \exists u \in \meaningof{A}, v \in {B}, t' \in \mathcal{T} . t \rightarrow t', (v u) \rightarrow t'\}$
  \item $\meaningof{ A 2 B } = \{ t \in \mathcal{T} | \exists u \in \meaningof{A}, v \in \meaningof{B}, t' \in \mathcal{T} . v \rightarrow t', (t u) \rightarrow t'\}$
  \item $\meaningof{ A 3 B } = \{ t \in \mathcal{T} | \exists u \in \meaningof{A}, v \in \meaningof{B}, t' \in \mathcal{T} . v \rightarrow t', (u t) \rightarrow t'\}$
etc.
\end{itemize}
